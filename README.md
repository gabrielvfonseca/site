## Features

- **Frontend**: Built with **Next.js** and **ui.shadcn**.  
- **Backend**: Content managed with **Contentlayer** and email integration via **Resend**.  
- **Infrastructure**: Hosted on **Vercel** with domain via **AWS Route 53**.  
- **Analytics**: Powered by **Vercel** for insights.  


## Installation

1. Clone the repository:

    ```bash
    git clone git@github.com:gabrielvfonseca/site.git
    cd next-site
    ```

2. Install dependencies:

    ```bash
    pnpm install
    ```

3. Run the development server:

    ```bash
    pnpm run dev
    ```


## Deployment

Hosted on **Vercel** with automated deployment from the `main` branch.
