I've been building (<i>complexier</i>) software over 6 years and have 
experimented with a variety of tools and technologies. 
Here's a detailed look into the software and tools I use daily:

### Always with me
- Macbook Pro M1 Max 16" (2020)
- iPhone 13 

### Productivity

- **Note-taking**: I use <a href="https://notion.so" target="_blank">Notion</a> for 
    organizing my thoughts, tasks, and projects.

- **Task Management**: I rely on <a href="https://superlist.com" target="_blank">Superlist</a> 
    for managing my daily tasks and projects.

- **Calendar**: I use <a href="https://fantastical.com" target="_blank">Fantastical</a> 
    for managing my calendar and scheduling events.

- **Focus**: I use <a href="https://opal.so" target="_blank">Opal</a> for blocking 
    distracting websites and apps during work hours.

### Development Tools

- **Code Editor**: I use <a href="https://code.visualstudio.com" target="_blank">Visual Studio Code</a>
    for all types of development tasks. It's fast, extensible, and has a vibrant 
    ecosystem of extensions.

- **Terminal**: It may sound old-school, but I prefer the macOS default 
    Terminal for most of my command-line tasks.

### Frontend Development

- **Framework**: I work with <a href="https://nextjs.org" target="_blank">Next.js</a> 
    for all types of projects. Absolutely my go-to choice!

- **Styling**: I use <a href="https://tailwindcss.com" target="_blank">TailwindCSS</a> 
    for rapid prototyping and styling components.

- **Component Library**: I use <a href="https://ui.shadcn.com" target="_blank">ui.shadcn</a> 
    for building accessible and responsive UI components.

### Backend Development

- **Server**: I work with <a href="https://nodejs.org" target="_blank">Node.js</a> 
    for server-side development.

- **Database**: I use primarily <a href="https://www.mongodb.com" target="_blank">MongoDB</a> 
    for NoSQL databases and <a href="https://www.postgresql.org" target="_blank">PostgreSQL</a> 
    for relational databases.

- **Deployment**: I deploy applications on <a href="https://vercel.com" target="_blank">Vercel</a>, 
    <a href="https://www.azure.com" target="_blank">Azure</a>,
    and <a href="https://aws.amazon.com" target="_blank">AWS</a>

### Design and Prototyping

- **Design Tools**: I work with [Figma](https://www.figma.com) for UI/UX design and prototyping.

- **Management**: I use <a href="https://linear.app" target="_blank">Linear</a> for managing 
    design tasks and projects.

---

### Programming Languages

- **C/C++**: I have experience with C/C++ for low-level programming tasks.

- **JavaScript**: I work with JavaScript for frontend and backend development.

- **TypeScript**: I use TypeScript for type-safe JavaScript development.

- **Python**: I use Python for scripting and data analysis tasks.

- **Rust**: I'm learning Rust for systems programming and performance-critical tasks.

- **SQL**: I work with SQL for database management and querying.
